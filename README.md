# tawq.in Community Microsite

A bilingual (English / Arabic) story-driven microsite for [tawq.in](https://tawq.in) — a platform enabling communities to add human audio narration to video in minority and endangered languages.

## About

This microsite is designed for indigenous and oral-language communities, particularly Jibbali/Hakli speakers of Dhofar, Oman. It explains what tawq.in is, how it works, and why communities control their own voice and data.

## Features

- Full English / Arabic (RTL) toggle
- Mobile-first, no dependencies
- Single self-contained `index.html`
- Modular structure for A/B audience variants

## Deployment

Hosted via GitHub Pages at:
`https://hytra3.github.io/tawqin_micro`

Or with custom domain: `community.tawq.in`
